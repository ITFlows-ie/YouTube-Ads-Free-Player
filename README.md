# YouTube Ads Free Player

A web application for watching YouTube videos ad-free with queue functionality.

## Deploying to GitHub Pages

This site is configured for automatic deployment to GitHub Pages.

### Setup Instructions:

1. **Enable GitHub Pages in repository settings:**
   - Go to Settings ‚Üí Pages
   - Under "Source", select "GitHub Actions"
   
2. **Automatic deployment:**
   - After merging changes to the `main` branch, the site will automatically update
   - You can also manually trigger deployment from the Actions tab

3. **Accessing the site:**
   - After successful deployment, the site will be available at: `https://itflows-ie.github.io/YouTube-Ads-Free/`
